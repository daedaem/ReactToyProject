# 리액트 토이프로젝트

## 1. MealOrder

### 주제

#### 음식을 주문하는 웹사이트

### 기술 스택

- React.js
- Redux Tool Kit
- Firebase

### 구현 기능

**1. 음식 리스트**

- firebase API 연동을 통한 초기 음식 주문 가능 리스트 화면
- 주문하는 음식 종류별, 갯수별 추가 기능

**2. 장바구니**

- 추가된 음식 목록 확인 기능
- 추가된 음식 종류별, 갯수별 추가/수정/삭제 기능
- firebase API 연동을 통한 장바구니 음식 리스트 추가
